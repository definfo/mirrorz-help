### pmbootstrap

通过 `pmbootstrap config` 更新镜像配置。

<CodeBlock>

```bash
pmbootstrap config mirrors.alpine {{http_protocol}}{{host}}/alpine/
pmbootstrap config mirrors.pmaports {{http_protocol}}{{mirror}}/
pmbootstrap config mirrors.systemd {{http_protocol}}{{mirror}}/extra-repos/systemd/
```

</CodeBlock>

其中 `mirrors.alpine` 部分也可前往 [Alpine 帮助](/alpine/) 选择其他的镜像站。

### 移除镜像配置

```bash
pmbootstrap config -r mirrors.alpine
pmbootstrap config -r mirrors.pmaports
pmbootstrap config -r mirrors.systemd
```
